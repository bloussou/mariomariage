## Install 

- Install haxe on your computer [download](https://haxe.org/download/)
- Run `haxe build.hxml`
- if cors error :
    1. install python3
    2. run `python3 -m http.server` in this folder
    3. open `http://0.0.0.0:8000/src/` in your browser web
